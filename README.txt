Arquivos:
- almoxpert_versao_final_nova.py
- requirements.txt
- render_start.txt
- static/logo_almoxpert.png

Como usar:
1. Substitua no GitHub o arquivo almoxpert_versao_final_nova.py por este novo.
2. Mantenha requirements.txt e a pasta static.
3. No Render, use:
   Build Command: pip install -r requirements.txt
   Start Command: python almoxpert_versao_final_nova.py
4. Faça Manual Deploy > Deploy latest commit.
